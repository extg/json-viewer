# Chromium DevTools Json Viewer

## Getting started

```sh
yarn
yarn bootstrap
```


## Build

```sh
yarn build
```

Build favicon

```sh
npx png-to-ico packages/json-viewer/public/icons/icon-48.png > packages/json-viewer/public/favicon.ico
```
